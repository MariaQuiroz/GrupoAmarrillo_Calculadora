# GrupoAmarillo_Calculadora

Este repositorio contiene el desarrollo de una **calculadora básica en C++**, realizada como proyecto colaborativo utilizando **Git y GitHub** para el control de versiones.

El programa permite realizar las siguientes operaciones:
- Suma
- Resta  
- Multiplicación  
- División  

## Tecnologías utilizadas
- Lenguaje: C++
- Control de versiones: Git
- Repositorio remoto: GitHub
- Entorno de desarrollo sugerido: Visual Studio / Visual Studio Code

## Organización del proyecto
El proyecto fue desarrollado de manera colaborativa utilizando las ramas suma, resta, multiplicacion, division y menu.

- Cada integrante trabajó una operación en una rama independiente.
- Posteriormente, el código fue integrado en la rama main.
- La versión final del programa se encuentra unificada en un solo archivo .cpp

Este enfoque permitió trabajar en paralelo sin conflictos y mantener un historial de cambios.

##¿Cómo descargar el programa?

### Opción 1: Desde GitHub
1. Ingresar al repositorio en GitHub.
2. Hacer clic en el botón Code.
3. Seleccionar Download ZIP
4. Descomprimir el archivo en la computadora.

### Opción 2: Usando Git

En el terminal (bash), se coloca el siguiente comando:
git clone https://github.com/MariaQuiroz/GrupoAmarrillo_Calculadora.git
Con esto, se clona el repositorio con los archivos del programa en la carpeta seleccionada para su almacenamiento.
